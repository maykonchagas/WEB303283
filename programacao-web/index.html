Listas

Esta é um lista não ordenada, representada por "ul"

  • Adicionado uma lista do tipo "ul" (sem ordem) na HTML.
  • Cada item é adicionado utilizando o elemento "li"
  • Os elementos não precisam estar ordenados

Esta é um lista ordenada, representada por "ol"

  1. Notem, agora a tag "ol" está abrindo uma lista ordenada.
  2. Mas os item das listas "li" são os mesmos.
  3. E também, perceba como os números são incrementados dentro da lista ordenada.
