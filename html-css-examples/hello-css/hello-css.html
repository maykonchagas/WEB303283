Olá, CSS

A CSS nos permite adicionar estilo nos elementos da HTML. Também tem outra página associada a esse exemplo.

Estilos de listas

Você pode adicionar estilos às listas não ordenadas com os seguintes marcadores:

  • discos
  • círculos
  • quadrados

e você pode ordenar listar com os seguintes números:

  1. decimais
  2. romanos-minúsculos
  3. romanos-maiúsculos
  4. alfabéticos-minúsculos
  5. alfabéticos-maiúsculos
  6. (e muitos outros)
